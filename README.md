## Projeto em destaque &middot; Featured project

### Valore — Wealth Management Desktop Application

Aplicação desktop de gestão de patrimônio desenvolvida para a **ANF Luxembourg S.A.**, empresa regulada pela **CSSF** (Commission de Surveillance du Secteur Financier).

**Stack técnico**
- Aplicação desktop em **Node.js + Tauri** com frontend web embarcado
- Sistema de identidade visual próprio com paleta dourada (`#C8A84B` em modo escuro / `#7A6020` em modo claro)
- Conjunto completo de ícones SVG/PNG customizados e navegação multi-seção

**Contexto regulatório**
- Desenvolvido considerando os requisitos aplicáveis em **MiFID II**, **KYC** e **GDPR**
- Atenção a rastreabilidade, segurança e tratamento de dados sensíveis

**Por que importa**
> Construir software num contexto financeiro supervisionado pela CSSF exige disciplina em segurança, rastreabilidade e qualidade — habilidades transferíveis a qualquer projeto técnico crítico, financeiro ou não.
